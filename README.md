## 介绍
QuarkUI 是一个可以通过json配置组合前端页面的低代码引擎；它提供丰富的组件，能帮助您通过很少的配置就能输出各式各样的页面。

## 开始

安装依赖

```bash
$ yarn
```

启动服务

```bash
$ yarn start
```
